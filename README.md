# Election: Run For President

A static website for the educational board game-style video game about the United States Electoral College.

## About the Game

Election: Run For President is an engaging educational game that teaches players about the Electoral College system through strategic gameplay.

## Website Structure

- `index.html` - Homepage with game information and download links
- `privacy.html` - Privacy policy page (required for iOS App Store)
- `images/` - Game assets and logos (managed with Git LFS)

## Development

This is a static website hosted on GitHub Pages.

To view locally, simply open `index.html` in a web browser.

## Deployment

The site is automatically deployed to GitHub Pages from the main branch.
